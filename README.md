 Sales & Profit Dashboard (Excel Project)
Project Overview
This project is part of the Future Interns Data Science program. It focuses on analyzing sales performance using the Superstore dataset and building an interactive Excel dashboard to extract meaningful business insights.
The goal is to transform raw transactional data into clear, actionable insights that support data-driven decision-making. 
Dataset
- Superstore Sales Dataset (Kaggle)
- ~9,994 transaction records
- Includes data on:
  - Sales
  - Profit
  - Discount
  - Region
  - Category
  - Sub-category
  - Customer segments
Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Data Cleaning & Transformation
 Dashboard Features
The Excel dashboard includes:
- Sales and Profit analysis by Region
- Category and Sub-category performance breakdown
- Profit vs Sales comparison
- Top-performing products
- Time-based sales trends
 Key Insights
- Furniture category (especially Tables) is generating losses due to high discounts
- Technology is the most profitable category with strong margins
- West region shows the highest overall profitability
- Sales increased significantly over the years, but profit efficiency needs improvement
 Key Learning
This project demonstrated that high sales do not always equal high profit. Proper cost and discount management is essential for sustainable business growth.
